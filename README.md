# 🏥 Disease Prediction Model

This project uses Machine Learning algorithms to predict diseases based on patient medical data such as age, symptoms, and medical test results.

---

## 🚀 Algorithms Used

* Logistic Regression
* Random Forest

---

## 📊 Dataset

* Medical dataset (e.g., Heart Disease / Diabetes)
* Features include age, symptoms, and other health indicators

---

##  Features

* Data preprocessing
* Feature selection
* Model training and testing
* Accuracy evaluation

---

##  Results

The model predicts the likelihood of disease with good accuracy using classification techniques.

---

##  How to Run

1. Install required libraries
2. Open the notebook (.ipynb)
3. Run all cells
